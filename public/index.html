TCP 2000  →  Form Services API + Dashboard
TCP 2001  →  PeerJS signaling server
TCP/UDP 3478  →  coTURN STUN/TURN
TCP 5349  →  coTURN TLS (for strict firewalls)

For school/enterprise firewalls: all traffic routes through port 443 via TURN relay.
